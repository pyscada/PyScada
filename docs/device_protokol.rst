- 15: `MeterBus (MBus) <https://github.com/pyscada/PyScada-MeterBus>`_
- 8X: Custom Worker
- 93 `Reserved (Serial) <https://github.com/clavay/PyScada-Serial>`_

- 96: Reserved (Event)
